# Style samples

Drop previous RM letters (one per `.md` file) into this folder.

When the report generator runs, every file here is added to the prompt
as a few-shot example so the LLM mimics the writer's tone, paragraph
rhythm, and Section 3 specificity.

Strip out client names, account numbers, and PII before uploading —
the demo deployment trusts you on this.

The Config tab in the app uploads files here for you.
